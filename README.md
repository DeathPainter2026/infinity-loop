# ∞ Infinity Loop — Media Tracker

## Як запустити

Просто відкрий `index.html` у браузері (Chrome або Firefox).

## Логін

- **Адмін:** DeathPainter / Reckless2015
- Гостьові акаунти — створюються в Налаштуваннях ⚙

## Теми

Перемикаються прямо в застосунку:
- 🐙 Ктулху (за замовчуванням) — темно-бірюзовий
- 🌑 Класик — темно-фіолетовий
- ⚡ Кіберпанк — неоновий рожево-пурпурний
- 💀 Жахи — темно-червоний

## OMDB API (автозаповнення IMDb)

1. Зайди на https://www.omdbapi.com/apikey.aspx
2. Обери Free (1000/day) → введи email → отримай ключ
3. У застосунку: ⚙ Налаштування → OMDB API → встав ключ

## Файли

```
infinity-loop/
  index.html          # Головна сторінка
  css/
    main.css           # Основні стилі
    theme-cthulhu.css  # 🐙 Тема Ктулху
    theme-classic.css  # 🌑 Класик
    theme-cyberpunk.css # ⚡ Кіберпанк
    theme-horror.css   # 💀 Жахи
  js/
    data.js            # База даних (localStorage)
    auth.js            # Авторизація
    ui.js              # Відображення списку
    analytics.js       # Аналітика
    app.js             # Ініціалізація
```

## Дані

Всі дані зберігаються в localStorage браузера.
- На одному пристрої/браузері — дані зберігаються між сесіями
- З іншого браузера або телефону — дані незалежні (поки не підключено Supabase)

## Наступний крок (хостинг)

Для доступу з будь-якого пристрою — Vercel + Supabase.
Інструкцію надам окремо.
